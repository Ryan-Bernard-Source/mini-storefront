Mini-Storefront

A small React + Next.js storefront web app where shoppers can browse items, filter by category and price, manage a cart, and watch inventory update automatically.

## Getting Started

Download ZIP Folder and extract or run:

```
git clone https://github.com/<your-username>/mini-storefront.git
cd mini-storefront
```

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features
- Browse and filter products
- Add/remove items from the cart
- View cart totals and item count
- Clear the cart (Restores inventory)
- Automatic stock updates
- Loading, error, and empty-state messages
- Tailwind CSS styling
